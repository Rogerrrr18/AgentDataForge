# 任务：Canvas 粒子烟花

用**单个 `index.html` 文件**（内联 JS，可用 Canvas 2D）实现一个交互式粒子烟花。

## 需求

1. **点击**画布任意位置发射一发烟花：从底部上升，到点击点附近**爆炸**成大量粒子。
2. 爆炸粒子沿各方向发散，受**重力**影响下落，带**拖尾**效果，颜色随时间**衰减/淡出**。
3. **空闲时自动循环**播放烟花（即使不点击，每隔一段时间自动放一发），保证页面始终有动效。
4. 配色丰富、有夜景氛围（建议深色背景），视觉有冲击力。
5. 单文件、零外部依赖；性能流畅（粒子数量适中，不掉帧）。

## 交付

把成品写到 `deliverables/index.html`（双击即可在浏览器打开运行）。

## 评分会检查（rubric）

- canvas 存在、无 console error
- 画面随时间变化（在动）
- 点击后画面变化（点击有响应、发射新粒子）
- 代码含重力 / 粒子数组 / 角度发散（源码级检查）
- 视觉裁判打分：配色、动效流畅度、审美冲击力
