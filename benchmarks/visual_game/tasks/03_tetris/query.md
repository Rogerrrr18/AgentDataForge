# 任务：俄罗斯方块

用**单个 `index.html` 文件**（内联 JS，Canvas 2D）实现一个完整的俄罗斯方块。

## 需求

1. **7 种标准方块**（I/O/T/S/Z/J/L），从顶部下落。
2. **方向键**：← → 左右移动，↑ 旋转，↓ 加速下落，空格硬降（可选）。
3. **旋转**正确（旋转矩阵，不越界、不重叠）。
4. **消行**：填满一整行后消除，上方下移；**计分**（消行越多分越高）。
5. **下一块预览** + 当前分数显示。
6. **game over**：堆到顶部时结束，可重新开始。
7. 单文件、零外部依赖；布局清晰（棋盘 + 预览 + 分数）。

## 测试钩子（为自动评分，必须实现）

请在全局暴露游戏状态，供 rubric 读取：

```js
window.__GAME = {
  board: <二维数组，行×列，0 表示空，非 0 表示方块颜色/类型>,
  score: <number>,
  over: <boolean>,
};
```

`board` 需随游戏状态实时更新（每次渲染后同步）。**未实现会被判 PARTIAL（降置信）。**

## 交付

把成品写到 `deliverables/index.html`（双击即可在浏览器打开运行）。

## 评分会检查（rubric）

- canvas 存在、无 console error
- `window.__GAME` 暴露（指令遵循）
- 按方向键后棋盘状态变化（移动/旋转）
- 旋转后占用格集合改变（旋转有效）
- 无输入时方块随时间下落（游戏循环）
- 视觉裁判打分：构图清晰度、审美
